# mfe-playground

**Install deps.**
yarn ci


**Start react shell app:** \
cd packages/react-mfe-shell \
yarn start


**build & serve mfe app 1:** \
cd packages/react-mfe \
yarn build-systemjs \
yarn server



**build & serve mfe app 2:** \
cd packages/react-mfe-1 \
yarn build \
yarn server


**Start ember shell app:** \
cd packages/ember-mfe-shell \
yarn start
